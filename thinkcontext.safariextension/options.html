Welcome to thinkContext, the web browser extension which adds contextually relevant, politically progressive information to your web browsing. This is the options page for the software running on your computer.

As you browse the web thinkContext consults progressive data sources and if it finds anything relevant to what you are looking at it will unobtrusively add that information right into the page. Look for the icons shown next to the campaigns below and hover your mouse over them to show more information. Subscribe to campaigns below.

Important note for Safari users! Make sure you set Safari to automatically update extensions so that when we add new features you get them right away. Open up Safari's preferences (from the "Safari" menu list in the upper left for Mac or from the "Edit" or gear menu items on Windows), then select the "Extensions" item from the list across the top. Then click on "Updates" from the lower left and then check the check box next to "Install updates automatically". See this screenshot.
